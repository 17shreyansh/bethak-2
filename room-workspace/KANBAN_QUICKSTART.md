## Get Started in 3 Steps

### 1. Run the Application
```bash
cd room-workspace
npm run dev
```

### 2. Login or Signup
- Navigate to `http://localhost:5173`
- Login with existing account or create a new one

### 3. Access the Kanban Board

**Option A: Demo Page (Recommended for Testing)**
- Go to: `http://localhost:5173/kanban-demo`
- Your personal card will be created automatically
- Try dragging, adding tasks, and editing!

**Option B: Room Workspace**
- Go to: `http://localhost:5173/rooms`
- Click "Open" on any room
- Click the "Kanban Board" tab
- Start managing your tasks!

## Features at a Glance

| Feature | Description | Permission |
|---------|-------------|------------|
| Personal Card | Each user gets their own card | Auto-created |
| Drag & Drop | Move cards between columns | Owner only |
| Edit Card | Change title/description | Owner only |
| Add Tasks | Create task checklist | Owner only |
| Complete Tasks | Mark tasks done | Owner only |
| View Cards | See all team members' cards | Everyone |

## Tips for Success

1. **Break Down Work**: Add multiple small tasks instead of one big task
2. **Update Status**: Move your card as you progress
3. **Be Descriptive**: Use clear titles and descriptions
4. **Check Progress**: Watch your task completion percentage
5. **Team Visibility**: Remember, everyone can see your card!

## Keyboard Shortcuts

- `Esc` - Cancel editing
- `Enter` - Save/Submit (in text fields)
- `Tab` - Navigate between fields

## Troubleshooting

**Card not appearing?**
- Make sure you're logged in
- Refresh the page
- Check that you're in a room or demo page

**Can't drag card?**
- Only you can drag your own card
- Make sure you're clicking the grip handle (â‹®â‹®)

**Can't edit card?**
- You can only edit your own card
- Other members' cards are read-only
